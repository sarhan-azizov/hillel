# Hillel Javascript full stack educational program

## Description

The Hillel project created for learning and improve javascript knowledge.

## Installation
The project based on docker. so, don't forget install it.

```bash
$ npm install
$ npm run seed:run
```

## Running the app

```bash
# development
$ npm run docker:up
$ npm run start

# watch mode
$ npm run docker:up
$ npm run start:dev

# production mode
$ npm run docker:up
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
